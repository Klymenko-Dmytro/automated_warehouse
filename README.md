# 📦 Automated Warehouse (WMS)

Система автоматизованого керування товарами на складі з функцією адресного зберігання, інтерактивною 2D-картою та живою симуляцією процесів у реальному часі.

## 🚀 Основні функції
*   **Адресне зберігання**: Кожна комірка має чіткі координати: Зона (Відділ), Ряд, Полиця, Місце.
*   **Інтерактивна 2D-карта**: Візуалізація складу "вигляд зверху" з масштабованими мікро-комірками та технологічними проходами.
*   **Розумний пошук**: Миттєвий пошук товарів за назвою або штрихкодом із динамічним підсвічуванням комірки на карті.
*   **Контрольна панель (AJAX)**: Ручне розміщення та вилучення товарів в один клік без перезавантаження сторінки.
*   **Жива симуляція подій**: Фоновій скрипт-генератор, який демонструє роботу складу в реальному часі.
*   **Live Журнал операцій**: Віконце логування, що відображає поточні дії (автоматичні та ручні).

## 🛠️ Стек технологій
*   **Backend**: Python, Django
*   **Database**: SQLite3
*   **Frontend**: HTML5, CSS3 (Flexbox/Grid), JavaScript (Fetch API / AJAX)

## 💻 Як запустити проект локально

### 1. Клонування репозиторію та налаштування
```bash
# Створення та активація віртуального середовища
python -m venv venv
# Для Windows:
venv\Scripts\activate
# Для macOS/Linux:
source venv/bin/activate

# Встановлення Django
pip install django
```

### 2. Підготовка бази даних
```bash
# Застосування міграцій
python manage.py migrate

# Генерація початкової сітки складу (виконати один раз)
python init_warehouse.py
```

### 3. Запуск системи
Для повноцінної демонстрації проекту потрібно відкрити **два паралельних вікна терміналу** в PyCharm:

*   **Термінал 1 (Веб-сервер)**:
    ```bash
    python manage.py runserver
    ```
    Після цього відкрийте браузер за адресою: `http://127.0.0`

*   **Термінал 2 (Жива симуляція для презентації)**:
    ```bash
    python run_simulation.py
    ```

## 📋 Структура адреси комірки
Кожна комірка на карті підписана у форматі: `Зона-Ряд-Полиця-Місце` (наприклад, `A-2-1-4`). При наведенні курсору на будь-яку комірку спливає підказка з повною інформацією про товар та штрихкод.
